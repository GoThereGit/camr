How to Participate

  • All you need is an Entry Form.
  • Participants must sign the LDC Evaluation License Agreement for the data/corpora used in the CAMRP 2023 Evaluation Task will be authorized via LDC only:
    • Each team should designate a member as the data contact person.
    • Each data contact person must fill in the LDC Evaluation License Agreement and send a signed copy of the agreement to ldc@ldc.upenn.edu.
    • Participants must use the received data/corpora in the CAMRP 2023 Evaluation Task only.
    • For details, please refer to the LDC Evaluation License Agreement.
  • Technical Reports/System Description Papers:
    • can be written in either Chinese (if you like) or English (ChatGPT doesn't count).
    • should be formatted according to CCL 2023 template.
    • should include at least the following sections: introduction, evaluation results, result analysis and references.
  • Don't miss the deadline, please😭
